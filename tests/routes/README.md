# Route demos

This test builds a tiny synthetic `cicpy` design and renders route examples with `SvgPrinter`.

Generated outputs:
- `ROUTE_DEMOS_svg/ROUTE_STRAIGHT.svg`
- `ROUTE_DEMOS_svg/ROUTE_L_VERTICAL.svg`
- `ROUTE_DEMOS_svg/ROUTE_ORTHOGONAL.svg`
- `ROUTE_DEMOS_svg.html`

Run it with:

```bash
make -C tests/routes test
```
